# Dental Clinic Management System
## Description
This is a Python GUI application built using **Python 3.13**, **PyQt**, and **Oracle Database**.
It manages patients, dentists, appointments, treatments, and billing.

## Technologies Used
- Python 3.13
- PyQt (GUI)
- Oracle Database
- cx_Oracle (or python-oracledb)
